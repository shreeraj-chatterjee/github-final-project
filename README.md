# Simple Interest Calculator

A simple bash script to calculate simple interest given the principal, rate of interest, and time period in years.

## Introduction

This script accepts user input for:
- **Principal** (P): The initial amount of money
- **Rate** (R): Annual rate of interest (in %)
- **Time** (T): Time period in years

It then outputs the Simple Interest using the formula: SI = (P * R * T) / 100

## Usage

```bash
bash simple-interest.sh
```
